# Contributing

Thanks for improving FinLens Catalyst Mapper.

## Contribution rules

- Keep the plugin public-safe.
- Do not add local absolute paths.
- Do not commit private API keys, endpoints, tokens, or personal provider bindings.
- Treat private or premium data sources as optional adapters, not hard dependencies.
- Do not claim real-time access unless the adapter actually exists and is documented.
- Keep timestamp and source validation rules intact for prices, macro data, policy releases, rates, FX, and valuation data.

## Scope for pull requests

Good contributions include:

- Better analytical frameworks
- Better validation rules
- Better output structure
- Clearer documentation
- Optional public-safe adapters

Please avoid bundling unrelated changes into a single pull request.
