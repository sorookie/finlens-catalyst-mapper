# Security

If you discover a security issue in this repository, please report it privately to the maintainer rather than opening a public issue with exploit details.

## Do not commit

- API keys
- Tokens
- Secrets
- Local absolute paths
- Private endpoints
- Personal broker or data-provider credentials

## Scope note

This plugin is designed to stay public-safe. Optional private adapters should live in a separate private layer and should not be merged into this public repository by default.
